  ### 💡 How Anyone Can Run It

  From any terminal prompt:

    # Standard run (uses default $HOME/src/kube-agents and today's date)
    ~/src/kube-agents-security-analysis/run_security_analysis.sh

    # Run for a custom date
    ~/src/kube-agents-security-analysis/run_security_analysis.sh 2026-07-15

    # Run for another user or a custom codebase location
    SRC_DIR=/path/to/src KUBE_AGENTS_DIR=/path/to/custom-repo ~/src/kube-agents-security-analysis/run_security_analysis.sh
